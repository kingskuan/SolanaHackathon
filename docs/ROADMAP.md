# Chain Will — Roadmap & Milestones

Short term (MVP - hackathon)
- Contract: Deposit, CreateWill, ProposeExecute, FinalizeExecute, Cancel
- Agents: Monitor (inactivity detection), Relayer (job execution)
- UI: basic create + deposit + show status
- Tests: unit tests for contract + integration script on devnet
- Submission: X Article + demo video + repo

Mid term (post-hackathon)
- Relayer marketplace w/ reputation & bond
- Support SPL tokens & NFTs
- Social recovery & guardians
- Onchain audit trail & analytics dashboard

Security considerations
- Challenge windows, guardian multi-sig, relayer bond
- Rate limit will creation & require small fees to prevent spam
